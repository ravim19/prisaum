Opens a page section with an uppercase eyebrow (leaf rule), bold title and optional intro line.

```jsx
<SectionHeading eyebrow="What we offer" title="Vocational Services"
  intro="Hands-on, certified training that builds independence." />
<SectionHeading align="center" size="lg" title="Whom We Serve" />
<SectionHeading onDark eyebrow="Get involved" title="Become a volunteer" />
```

Props: `align` (`left`/`center`), `size` (`sm`/`md`/`lg`), `onDark` for forest-green sections, `as` to set the heading level.
