Signature tile for a vocational or therapeutic programme — icon chip, title, description and optional link.

```jsx
<ProgramCard
  icon="https://prisaum.com/hs-fs/hubfs/vocational-training.png"
  title="Baking & Confectionery"
  description="Hands-on training in baking, decoration and safe kitchen practice."
  tag={<Tag tone="leaf">Vocational</Tag>}
  href="/programs#baking" />
```

Pass `icon` as a brand spot-icon URL or an inline SVG. Set `href` to make it a hover-lifting link with a "Learn more" arrow.
