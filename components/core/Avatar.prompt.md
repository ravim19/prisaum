Circular avatar for volunteers, benefactors and staff; shows initials on a soft-green chip when no image is given.

```jsx
<Avatar name="Pauline Wright" src="/people/pauline.png" size="lg" ring />
<Avatar name="James Stewart" />
```

Sizes: `xs` / `sm` / `md` / `lg` / `xl`. Add `ring` for a soft green halo (used on benefactor cards).
