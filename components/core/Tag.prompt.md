Pill label for program categories ("Vocational", "Therapeutic"), admission status, and metadata chips.

```jsx
<Tag tone="leaf">Vocational Training</Tag>
<Tag tone="success" dot>Admissions Open</Tag>
<Tag tone="warning" variant="solid">NIOS</Tag>
```

Tones: `leaf` (brand), `success`, `warning`, `danger`, `info`, `neutral`. Variants: `soft` (default), `solid`, `outline`. Add `dot` for a live status indicator; `icon` for a leading glyph.
