Warm surface container for program tiles, info panels, testimonials and stat blocks.

```jsx
<Card interactive>
  <h4>Baking & Confectionery</h4>
  <p>Hands-on training in baking, decoration and packaging.</p>
</Card>
<Card tone="leaf" padding="lg">…</Card>
```

Tones: `default` (white), `leaf` (tinted green panel), `dark` (forest section). `elevation`: `flat` / `card` / `raised`. Set `interactive` for hover-lift on clickable cards, `padding="none"` for full-bleed image cards.
